# 📚 Study Nook Backend

**Study Nook**, a study room booking platform that enables users to browse available rooms, make reservations, manage bookings, and experience a seamless study-space management system.

---

## 🚀 Features

* 🔐 User Authentication & Authorization
* 🏢 Study Room Management
* 📅 Room Booking System
* ✏️ Update Booking Information
* ❌ Cancel Bookings
* 📊 Booking Status Management
* 🔍 Fetch Room & Booking Details
* 🌐 RESTful API Architecture

---

## 🛠️ Technologies Used

* Node.js
* Express.js
* MongoDB Atlas
* CORS
* Dotenv

---

## 🔒 Security

* Environment Variable Protection
* CORS Configuration

---

## 🎯 Project Goal

The goal of Study Nook is to provide students and learners with a simple and efficient platform to discover, reserve, and manage study spaces while ensuring a smooth and secure booking experience.
